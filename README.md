# <a href="https://github.com/nlagasse/Music-Player/releases"><img width="32" height="32" alt="mlogoround" src="https://github.com/user-attachments/assets/b6354b40-8015-4f0d-9cff-fe76a1725f9d" /></a> Music Player for MP3/FLAC Audio Files and Albums

Music player inspired by Spotify for organizing and playing locally owned music on windows machines. Works with album folders which contain MP3/FLAC audio files (or a combination of both!) and cover art(s). Check out the releases for the latest version to download!

<img width="1918" height="1137" alt="musicplayer-v0 5 0" src="https://github.com/user-attachments/assets/919e6691-6afb-42c1-881d-c57018caf873" />

* Every album added needs at least one song (MP3 or FLAC) and one album art (PNG or JPG or JPEG) included!

Everything I used to make this project was entirely self taught and new to me, as I had no prior experience in developing a Windows desktop application from scratch, nor experience with WPF, C#, JSON databases, or UX.


## Controls: 

'Add' Button - Select the folder(s) containing your music and cover arts to be added to your library, no need to directly select each file individually, just the album folder will do!

'Play Liked Songs Only' Button - This button will only play liked songs of the currently playing album when toggled. If there are no songs liked in an album, nothing will happen! Get to liking!

Middle Mouse Button on an album in your library - This will instantly start playing the album, no need to left mouse button click and press a second button to start playing an album. 

Mouse scrolling on the entire player bar row - This will let you change the volume from anywhere on the bar, so you don't have to track down the volume slider and drag it. Unless you'd like to!
